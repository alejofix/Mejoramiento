NeuralJQueryScript Clase de Ayuda para la Creación de Script JQuery.

NeuralPHP Framework provee una que generar scripts prefabricados en JQuery de forma simple.


Parametros De Asignación


IdPrincipal('Id Principal')


Asigna el Id Principal para la asignación base del Script. 

NeuralJQueryScript::IdPrincipal('Id_Principal');


IdSecundario('Id Secundario')


Asigna el Id Secundario para la asignación base del Script. 

NeuralJQueryScript::IdSecundario('Id_Secundario');


Parametros('Matriz de Parametros')


Asigna la matriz de parametros de par Nombre => Valor 

NeuralJQueryScript::Parametros(array('Nombre' => 'Valor'));


Puntero('Puntero a Enviar')


Asigna el nombre del puntero de la petición a enviar 

NeuralJQueryScript::Puntero('Nombre_Puntero');


Tiempo('Duración')


Asigna la duración correspondiente 

NeuralJQueryScript::Tiempo(10);


URL('Dirección Web')


Asigna la duración correspondiente 

NeuralJQueryScript::URL('http://dominio.com');


Scripts Construidos


CargarIntervaloTiempo('Etiquetas Script', 'Libreria de JQuery')


Carga una petición a una URL con una duración predeterminada.

  1. Etiquetas Script: valor por defecto true, agrega las etiquetas script.
  2. Libreria de JQuery: valor por defecto false, agrega la libreria de JQuery.

Son obligatorios los Parametros de Asignación: IdPrincipal, URL y Tiempo. 

								/**
								 * Recordar que el metodo CargarIntervaloTiempo()
							 	 * debe ser el ultimo metodo de la cadena para generar
							 	 * el Script correspondiente.
								 */
								return NeuralJQueryScript::IdPrincipal('Id_Principal')
													->Tiempo(10)
													->URL('http://dominio.com')
													->CargarIntervaloTiempo();
