NeuralJQueryFormularioValidacion Clase de Ayuda para la Validación de Formularios con JQuery y el Plugin Validate.

NeuralPHP Framework provee una que generar scripts prefabricados en JQuery de forma simple.


NeuralJQueryFormularioValidacion('Etiquetas Script', 'Libreria Validate', 'Libreria JQuery')


Instancia genera las etiquetas script y las librerias correspondientes.

  1. Etiquetas Script: valor por defecto true, agrega las etiquetas de Script.
  2. Libreria Validate: valor por defecto true, agrega la libreria Plugin Validate.
  3. Libreria JQuery: valor por defecto false, agrega la libreria de JQuery.

$Validacion = new NeuralJQueryFormularioValidacion(true, true, false);


ArchivoExtension('Nombre del Campo', 'Matriz de MIME Type's', 'Mensaje')


Genera la Validación que la extensión del archivo sea el correcto.

  1. Nombre del Campo: la propiedad NAME del input.
  2. Matriz de MIME Type: es una matriz con el MIME Type de los archivos, estos MIME Type's se pueden consultar en esta página: Sitepoint.com
  3. Mensaje: Mensaje Personalizado del Error.

								/**
								 * 
								 * Mensaje Por Defecto si no se utiliza uno personalizado
								 * "Solo Son Aceptados Los Archivos con la Extensión Especificada"
								 */
								$Validacion->ArchivoExtension('Archivo', array('image/jpeg', 'application/x-compressed', 'application/msword'), 'Solo Son Aceptados los Archivos con Extensiones JPG, ZIP y DOC');


CampoIgual('Nombre del Campo', 'Id del Campo a Comparar', 'Mensaje')


Genera la Validación el campo sea igual a otro campo.

  1. Nombre del Campo: la propiedad NAME del input.

  3. Mensaje: Mensaje Personalizado del Error.

Remoto('Nombre del Campo', 'Url', 'Metodo de Envío', 'Asincrónico', 'Mensaje')


Genera la validación del campo enviando el dato sobre una petición Ajax retornado true o false.

  1. Nombre del Campo: la propiedad NAME del input.
  2. Url: dirección donde se enviara la petición AJAX.
  3. Metodo de Envío: Metodos soportados POST y GET.
  4. Asincrónico: es un valor true o false si se quiere de forma Asincróna o no.
  5. Mensaje: Mensaje Personalizado del Error.

								/**
								 * 
								 * Mensaje Por Defecto si no se utiliza uno personalizado
								 * "No Se Encuentra La Información Ingresada"
								 */
								$Validacion->Remoto('Nombre', 'http://dominio.com/', 'POST', false, 'No Se Encuentra La Información Ingresada');
								/**
								 * Esta validación es del Script que recibe la Petición Ajax
								 * Si el valor comparado al enviado son iguales se envia valor texto 'false'
								 * Si el valor comparado al enviado no son iguales se envia valor texto 'true'
								 */
								$Validacion = $_POST['Nombre'];
								return ($Validacion == 'Valor de Comparación') ? 'false' : 'true';

Ejemplo de Uso

Validación Normal

								$Validacion = new NeuralJQueryFormularioValidacion(true, true, false);
								$Validacion->Requerido('Nombre', 'El Campo es Requerido');
								$Validacion->Requerido('Apellido', 'El Campo es Requerido');
								$Validacion->Requerido('Telefono', 'El Campo es Requerido');
								$Validacion->Numero('Telefono', 'Debe Ingresar un Valor Numerico');
								return $Validacion->Constructor('IdFormulario');

Multiples Validaciones

								/**
								 * Pasamos el parametro 'Libreria JQuery' true
								 * Pasamos el Parametro 'Libreria Validate' true
								 */
								$Validacion_Form1 = new NeuralJQueryFormularioValidacion(true, true, true);
								$Validacion_Form1->Requerido('Nombre', 'El Campo es Requerido');
								$Validacion_Form1->Requerido('Apellido', 'El Campo es Requerido');
								$Validacion_Form1->Requerido('Telefono', 'El Campo es Requerido');
								$Validacion_Form1->Numero('Telefono', 'Debe Ingresar un Valor Numerico');
								echo $Validacion_Form1->Constructor('IdFormulario1');
								
								/**
								 * Pasamos el parametro 'Libreria JQuery' false
								 * Pasamos el Parametro 'Libreria Validate' false
								 * Ya que en este punto fueron llamadas desde la $Validacion_Form1
								 */
								$Validacion_Form2 = new NeuralJQueryFormularioValidacion(true, false, false);
								$Validacion_Form2->ArchivoExtension('Archivo', array('image/jpeg', 'application/x-compressed', 'application/msword'), 'Solo Son Aceptados los Archivos con Extensiones JPG, ZIP y DOC');
								$Validacion_Form2->CampoIgual('Password', 'Validar_Password', 'Las Contraseñas No Son Iguales ');
								$Validacion_Form2->Email('Nombre', 'El Formato del Correo No Es Correcto');
								$Validacion_Form2->Numero('Nombre', 'Debe Ingresar un Valor Numerico');
								$Validacion_Form2->ControlEnvio(
								    NeuralJQueryAjaxConstructor::TipoDatos('html')
								        ->TipoEnvio('POST')
								        ->Datos('#IdFormulario')
								        ->URL('http://midominio.com/')
								        ->AntesEnvio('$("#IdEspera").html("");')
								        ->FinalizadoEnvio('$("#IdRespuesta").html(Respuesta);')
								);
								echo $Validacion_Form2->Constructor('IdFormulario2');
