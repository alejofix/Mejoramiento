NeuralJQueryConstructor Clase de Ayuda para la Creación de Script JQuery.

NeuralPHP Framework provee una clase para generar scripts de JQuery de forma simple.



Inicializar('Etiquetas Script', 'Libreria JQuery')


Genera las etiquetas Script y agrega la libreria de JQuery.
Este metodo no es necesario en dado caso que se generen scripts sobre algunas funciones de otros scripts. Solo aceptan valores booleanos. 

NeuralJQueryConstructor::Inicializar(true, true);


DocumentoListo()


Genera el codigo $(document).ready(). 

NeuralJQueryConstructor::DocumentoListo();


AccionAparecer('Selector', 'Duración', 'Función')


Genera la transición de Aparecer, función JQuery: fadeIn().

  • Selector: Selector de la etiqueta html ya sea del #Id ó de la .Clase
  • Duración: Duración en segundos de la transición
  • Función: valor true para seguir agregando funciones de lo contrario valor false

								/**
								 * Manejar solo el Selector
								 * El tiempo se manejara por defecto 400ms
								 */
								NeuralJQueryConstructor::AccionAparecer('#IdDiv');
								
								/**
								 * Manejar el Selector y tiempo
								 */
								NeuralJQueryConstructor::AccionAparecer('#IdDiv', 3);
								
								/**
								 * Manejar solo el Selector, tiempo y una función adicional
								 */
								NeuralJQueryConstructor::AccionAparecer('#IdDiv', 3, true)->MostrarAlerta('"Se Muestra el DIV"');


AccionCargar('Selector', 'URL', 'Función', 'Parametros')


Genera la carga de una URL a través de una petición Ajax, función JQuery: load().

  • Selector: Selector de la etiqueta html ya sea del #Id ó de la .Clase
  • URL: url donde se enviara la petición correspondiente.
  • Función: valor true para seguir agregando funciones de lo contrario valor false.
  • Parametros: matriz de datos POST que se enviaran junto con la petición.

								/**
								 * Petición de solo carga
								 */
								NeuralJQueryConstructor::AccionCargar('#IdDiv', 'http://dominio.com/');
								
								/**
								 * Petición con envio de datos POST
								 */
								NeuralJQueryConstructor::AccionCargar('#IdDiv', 'http://dominio.com/', false, array('Nombre' => 'Valor'));
								
								/**
								 * Petición con envio de datos POST y función de confirmación de Finalización de Petición
								 */
								NeuralJQueryConstructor::AccionCargar('#IdDiv', 'http://dominio.com/', true, array('Nombre' => 'Valor'))->MostrarAlerta('"Se Cargo la Petición"');


AccionClick('Selector')


Genera la accion al generar click en el elemento seleccionado, función JQuery click().

  • Selector: Selector de la etiqueta html ya sea del #Id ó de la .Clase

NeuralJQueryConstructor::AccionClick('#IdDiv');


AccionDesaparecer('Selector', 'Duración', 'Función')


Genera la transición de Desaparecer, función JQuery: fadeOut().

  • Selector: Selector de la etiqueta html ya sea del #Id ó de la .Clase
  • Duración: Duración en segundos de la transición
  • Función: valor true para seguir agregando funciones de lo contrario valor false

								/**
								 * Manejar solo el Selector
								 * El tiempo se manejara por defecto 400ms
								 */
								NeuralJQueryConstructor::AccionDesaparecer('#IdDiv');
								
								/**
								 * Manejar el Selector y tiempo
								 */
								NeuralJQueryConstructor::AccionDesaparecer('#IdDiv', 3);
								
								/**
								 * Manejar solo el Selector, tiempo y una función adicional
								 */
								NeuralJQueryConstructor::AccionDesaparecer('#IdDiv', 3, true)->MostrarAlerta('"Desaparecio el DIV"');


AccionIntervaloTiempo('Duración')


Genera la carga de una función especifica continua en un intervalo según la duración ingresada, función JQuery: setInterval.

  • Duración: Duración en segundos del intervalo

NeuralJQueryConstructor::AccionIntervaloTiempo(10);


Css('Selector', 'Matriz de Propiedades')


Genera el cambio de estilos de la etiqueta seleccionada, función JQuery: css().

  • Selector: Selector de la etiqueta html ya sea del #Id ó de la .Clase
  • Matriz de Propiedades: Matriz asociativo con las propiedades css que se manejaran en el selector

NeuralJQueryConstructor::Css('#IdDiv', array('font-family' => 'verdana', 'color' => '#2E64FE'));


InsertarHTML('Selector', 'HTML')


Inserta las etiquetas HTML en la etiqueta del selector, función JQuery: html().

  • Selector: Selector de la etiqueta html ya sea del #Id ó de la .Clase
  • HTML: etiquetas y estructuras HTML, tambien puede insertarse texto o algún contenido diferente al HTML.

NeuralJQueryConstructor::InsertarHTML('#IdDiv', '
NeuralPHP Framework
');


MostrarAlerta('Texto a Mostrar')


Genera una alerta de JavaScript, función JQuery: alert().

  • Texto a Mostrar: Texto o procedimiento de JavaScript

								/**
								 * Alerta Simple
								 */
								NeuralJQueryConstructor::MostrarAlerta('"Esta es Una Alerta Simple"');
								
								/**
								 * Alerta Compuesta
								 */
								NeuralJQueryConstructor::MostrarAlerta('"Texto JavaScript " + Variable_JavaScript');


Ejemplos de Uso

Cargar Petición Ajax Haciendo Click en Hipervinculo

PHP

								return NeuralJQueryConstructor::Inicializar(true)
																->DocumentoListo()
																->AccionClick('#IdLink')
																->AccionCargar('#IdDiv', 'http://dominio.com/');

Cargar Petición En un Intervalo de Tiempo

PHP

								return NeuralJQueryConstructor::Inicializar(true)
																->DocumentoListo()
																->AccionCargar('#IdDiv', 'http://dominio.com/', true)
																->AccionIntervaloTiempo(10)
																->AccionCargar('#IdDiv', 'http://dominio.com/');
