NeuralCorreoSwiftMailer Clase de Ayuda para Envío de Correos Electrónicos.

NeuralPHP Framework provee una clase para generar procesos sobre el envio de correos electrónicos de forma simple.
Si desea más información al respecto puede ingresar al sitio web de SwiftMailer



NeuralCorreoSwiftMailer('MiApp', 'Prioridad', 'Tipo de Contenido', 'Codificación')


Se genera la instancia de la clase NeuralCorreoSwiftMailer la cual requiere los parametros:

MiApp: La aplicación configurada en el archivo de configuración de Correo.

Prioridad: Por defecto 3, La priodidad que tendra el correo envíado:

  1. 1: Urgente
  2. 2: Alta
  3. 3: Normal
  4. 4: Baja
  5. 5: Muy Baja

Tipo de Contenido: por defecto HTML, el tipo de contenido puede ser txt, html.
Codificación: por defecto UTF8, se puede especificar uno personalizado. 

$Correo = new NeuralCorreoSwiftMailer('MiApp', 3, 'html', 'UTF-8');


Asunto('Asunto del Mensaje')


Asigna el asunto del mensaje; 

$Correo->Asunto('Asunto del Mensaje');


EnviarA('nombre@correo.com')


Asigna el correo electrónico al cual se enviara el mensaje. 

$Correo->EnviarA('nombre@correo.com');


EnviarCopia('nombre@correo.com')


Asigna el correo electrónico al cual se le envíara una copia del mensaje 

$Correo->EnviarCopia('nombre@correo.com');


EnviarCopiaOculta('nombre@correo.com')


Asigna el correo al cual se le enviara una copia oculta del mensaje. 

$Correo->EnviarCopiaOculta('nombre@correo.com');


ArchivoAdjunto('Ubicación del Archivo')


Adjunta un archivo al mensaje, desde la ruta raiz de disco. 

								/**
								 * Ejemplo de Ubicación en Linux
								 */
								$Correo->ArchivoAdjunto('/opt/win/archivo.txt');
								
								/**
								 * Ejemplo de Ubicación en Windows
								 */
								$Correo->ArchivoAdjunto('C:\Mis_Documentos\archivo.txt');


MensajeAlternativo('Mensaje Alternativo')


Asigna el mensaje alternativo, este se maneja en dado caso que el mensaje principal sea de un formato que el cliente de correo ó el navegador no soporte.
Normalmente este texto alternativo es PLANO. 

$Correo->MensajeAlternativo('Mensaje Alternativo en Caso de no soporta HTML');


Mensaje('Mensaje HTML')


Asigna el mensaje principal, normalmente mensaje en formato HTML. 

$Correo->Mensaje('Mensaje HTML');


EnviarCorreo()


Genera el proceso del envío del Correo Electrónico. 

$Correo->EnviarCorreo();


Ejemplo de Uso

								$Correo = new NeuralCorreoSwiftMailer('MiApp', 3, 'html', 'UTF-8');
								$Correo->Asunto('Asunto del Mensaje');
								$Correo->EnviarA('nombre@correo.com');
								$Correo->EnviarCopia('nombre@correo.com');
								$Correo->EnviarCopiaOculta('nombre@correo.com');
								$Correo->ArchivoAdjunto('/opt/win/archivo.txt');
								$Correo->MensajeAlternativo('Mensaje Alternativo en Caso de no soporta HTML');
								$Correo->Mensaje('Mensaje HTML');
								$Correo->EnviarCorreo();
